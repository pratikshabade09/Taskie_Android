# 📱 Taskie – Smart Task Manager App

**Taskie** is a simple, smart, and user-friendly Android app that helps you manage your daily tasks efficiently.
It combines voice input, smart reminders, and an intuitive interface to make task management effortless.

---

## ✨ Features

* ✅ Add, edit, and delete tasks
* 🎤 Voice-based task creation (just speak your task)
* ⏰ Daily 6 AM summary notification
* 📌 Task prioritization (High, Medium, Low)
* 🏷️ Smart tagging system
* 🔄 Swipe to complete or delete tasks
* 📊 Clean and organized task view
* 🔔 Reminder system for tasks

---

## 🧠 How It Works

* You can **type or speak** a task
* The app **understands your input** and creates the task
* Tasks are stored locally in the database
* Every morning at **6 AM**, you receive a summary of your tasks
* You can mark tasks complete or manage them easily

---

## 🛠️ Tech Stack

* **Android (Java)**
* **MVVM Architecture**
* **Room Database**
* **WorkManager (for daily notifications)**
* **RecyclerView + DiffUtil**
* **SpeechRecognizer API**
